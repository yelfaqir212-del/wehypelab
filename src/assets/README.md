# Assets Folder

## Required Images

Please place the following images in this folder:

1. **founder1.jpg** - First founder image (will display in color)
2. **founder2.jpg** - Second founder image (will display in black and white)

### Image Requirements:
- Format: JPG or PNG
- Recommended size: 800x800px or larger
- Aspect ratio: Square (1:1) works best
- Quality: High resolution for best results

### Notes:
- The second founder image will automatically have a grayscale filter applied via CSS
- Images will be displayed in hexagonal frames with animations
- Make sure image filenames match exactly: `founder1.jpg` and `founder2.jpg`
